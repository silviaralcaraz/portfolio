# Personal-Portfolio

Simple web to use as personal portfolio: https://silviaralcaraz.github.io/personal-portfolio.

## Requirements

- nodejs
- npm

## Execute this project

"$ git clone https://github.com/silviaralcaraz/personal-portfolio/"
"$ cd personal-portfolio"
"$ npm install"
"$ npm start"
